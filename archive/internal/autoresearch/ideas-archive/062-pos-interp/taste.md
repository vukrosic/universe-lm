## r1 — 2026-06-11 — verdict: reject

- **Out-of-scope mechanism.** PI is a long-context *extension* technique: it downscales position indices so that *long* sequences map back into the *pretrained* rotary phase band. The screen harness is at `max_seq_len=2048` (per `configs/llm_config.py`) and the model is 0.94M params trained on 3M tokens. There is no "long context" to extend *from* — the model never had a long-range phase regime to drift out of, so there is no phase-drift failure to fix.
- **No operating point at tiny1m3m.** At `alpha=1.0` (no scaling) the mechanism reproduces the baseline bit-for-bit. At `alpha<1` it *compresses* indices inside the trained 0–2048 range, which is the wrong direction (we're already in-range, so shrinking the rotary phase band can only reduce effective resolution and harm perplexity). The "lever" has zero positive ground to stand on in this regime.
- **Information value is null.** The screen has no long-context evaluation branch. A `use_pos_interp` flag with default `alpha=1.0` is a no-op A/B; anything else is a guaranteed loss against the same baseline we already have. There is no clean null that teaches us anything about the recipe — the bet can't be made sharp at this scale.
- **Scale mismatch is total.** The "scale evidence" cites LLaMA 7B → 32k context. Tiny1m3m is ~7000× smaller on params and operates at ~16× shorter context with no extended-ctx test target. The taste prompt is explicit: *"An idea that only pays off at larger scale, or needs data/infra we don't have, has no taste here regardless of how good the paper is."* This is exactly that case.
- **No re-pitch can save it.** The only way to make this bet informative here would be to (a) add a long-context eval branch the harness doesn't have, (b) pre-train the screen at a shorter ctx then test at longer, and (c) demonstrate that the 0.94M model even *has* a long-range phase-coverage failure. None of that is "one sharp sentence of we expect X because Y" — it's a different screen.
- **Verdict:** `reject`. Long-context extension has no seat at tiny1m3m. Close; not a portfolio gap.
