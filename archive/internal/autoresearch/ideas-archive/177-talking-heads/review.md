Not a fabricated or misread citation. ✓
- **Mechanism is a mechanism, not an HP tune.** Two learnable H×H linear projections (pre-softmax `M`, post-softmax `M_out`), both initialized to identity ⇒ literal `M @ x = x` and `M_out @ x = x` ⇒ **byte-identical at step 0** (max-abs-diff = 0.0 vs baseline, no tolerance needed — it's a true mathematical identity, not a near-identity with 1e-3 slack). This is the strongest possible step-0-bit-identity claim. ✓
- **🔴 tiny1m3m / seed 42 only.** The protocol's spec is locked: run at tiny1m3m, seed 42, vs the cached baseline (6.4394±0.04 / 6.4447±0.0488). No reference to screen20m or larger tiers. ✓

Not a duplicate. ✓
