## r1 — 2026-06-15 — verdict: accept

- **Bet is sharp and distinct from the per-head null pile.** "Let each head learn its own window, init at full sequence, expect to beat the global SWA=512 winner because 512 globally is a compromise between heads that want short windows (local features) and heads that want long windows (long-range)." That is one crisp sentence; it is the right shape for this gate.
- **Mechanism is structurally different from 152/155/160/166**, which all acted on score *magnitudes* (bias / temperature / post-AV gain / additive RPE) and nulled because per-head scalar gradient signal is too weak at 0.94M/12L/4H. 182 acts on the spatial pattern of attention (which positions a head attends to), not on the score. A mis-tuned window has direct loss cost — narrow too far and you lose tokens, the gradient pushes wider; wide too far and the softmax is unchanged, no pressure. That is a fundamentally different gradient dynamic than a per-head gain near 1.0.
- **Clean null value.** Even a wrong-sign/inside-band result closes the per-head-WINDOW axis at 0.94M as a separate line from per-head-GAIN/BIAS/TEMP/RPE. The information is NOT redundant with the four prior per-head nulls because the mechanism shape differs.
- **Init spec is clean.** `w_h_init=10 ⇒ sigmoid≈0.99995 ⇒ W_h≈T ⇒ relu(|t−s|−T/2)=0` for all s≤t with s∈[0,T). Step-0 byte-identical at fp32, +48 params (+0.005%), no other knobs touched. This is the kind of clean ablation the definition gate can defend.
- **Leverage if it works.** Per-head window is a real inductive bias (BigBird / Longformer / per-head window in BigBird ablation). If a per-head window mix beats global 512 at tiny1m3m, this opens a per-head-window family for 135M where H=12+ gives the gradient more to specialize. The screen-plays-to-135M test is satisfied.
- **Transfer-risk: med is honest.** Windowed attention is validated at 100M+; per-head learnable window is novel at this scale, but the underlying locality prior is well-established. The risk is more "specialization needs more heads" (H=12+) than "mechanism is broken."
- **Pick one sub-lever for the run.** The design sketch proposes both hard window (W_h via sigmoid) and soft Gaussian decay (λ_h via (t−s)²). Run only the hard-window variant on r1; the soft-decay variant is a different mechanism and should be its own idea if the hard-window run is interesting.
- **Portfolio note.** This is the 5th per-head-attention-shape variant at 0.94M after 152/155/160/166/174. The accept holds because the lever shape differs (spatial pattern vs score magnitude), but the definition gate should not stack another per-head score-shape variant on top of this — diversify at the next queue.
