# Taste log — 189 cosformer-linear-attn

## r1 — 2026-06-15 — verdict: revise
- **Bet is a vibe, not a prediction.** "Null confirms softmax binds; win suggests cosine is closer to optimal" reads like an after-the-fact framing of both outcomes, not a sharp "we expect Δ ≤ X because Y" prediction. cosFormer's published LM evidence (Qin et al. 2022, arXiv:2202.08791) is *parity with softmax*, not a quality win at any scale — there is no published "cosine feature map beats softmax" data point to anchor the bet. Sharpen it: pick one mechanism-level prediction that distinguishes a cosine-kernel win from a softmax-binds null (e.g. "val_acc at step 50 ≥ 0.10 because the cosine kernel is more diffuse and generalizes better with few training tokens", or "early-step attention entropy > softmax's because φ(x)=[cos x, sin x] is non-vanishing on the full QK range"). Without a sharp prediction, the slot is wasted on a likely null.
- **Bit-identity claim is suspect, please re-derive.** With γ=0 init and Q,K ~ N(0, 0.02²) at step 0, cos(Q) ≈ 1 − Q²/2 ≈ 1, not Q. So Q'K'^T ≈ 1 (per head, per position) and the cosFormer normalization Q'·K'^T sums to ≈ T, giving *uniform* attention at step 0, not softmax. The miner's "max|Δ logits| < 1e-5 at step 0 with seed 42" needs an explicit numerical check, not a hand-wavy Taylor expansion. If γ=0 truly is bit-silent, show the math for why the normalization fixes it; otherwise, define a γ-init that *is* softmax-equivalent at step 0 (or accept the lever is not bit-identical and pick a sharper reference point — e.g. γ_init such that φ(K)·φ(K)≈K·K at small K, which is what softmax's `exp(QK^T/√d)` first-order Taylor reduces to).
- **Information value of a null is low vs the existing 148-focal-mod close.** 148 already closed the "non-softmax attention" axis at 0.94M. A null at 189 would extend the null to *kernel-feature-map* linear attention — a 3rd null in the broader "alternative attention shape" family (after 004-retnet retention Δ=+0.04 and 008-gated-deltanet taste-reject off-niche). A win would be a real finding but the prior probability is very low given the published cosFormer LM evidence. Frame the slot as "if win → we have a 125M-validated kernel shape that beats softmax at 0.94M, a major finding; if null → confirms softmax kernel binds at this tier, extending 148's null to linear-attention family." Make the post-null information value explicit, not implied.
- **Niche fit is OK on shape, not on leverage.** The lever is a mechanism (kernel reparameterization, not HP), zero-init-able via γ, and the kernel-shape bet CAN fire at 0.94M (T=2048 is too short for the linear-time complexity to matter, so the bet reduces to kernel comparison — the miner correctly identifies this). 154-rebased-attn showed the "attention shape" lane has real leverage at this tier (Δ=-3.4766 record break), so the family is not exhausted. But 154 was an additive rebase of K/V; 189 is a *replacement* of softmax with a different kernel — a much bigger change with much higher divergence risk. The transition risk argues for a *tighter* pass-bar, not a looser one.
- **Suggested re-pitch:** (1) one sharp mechanism-level prediction (val_acc trajectory, early-step entropy, or similar — pick one and quantify), (2) explicit step-0 numerical verification of the bit-identity claim with γ_init chosen for the purpose, (3) state the post-null information value relative to 148-focal-mod, (4) tighten the pass-bar to match the higher transition risk (the protocol's default |Δ|≤0.01 is too loose for a softmax-replacement lever).
