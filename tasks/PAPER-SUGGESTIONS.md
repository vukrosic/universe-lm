# Paper suggestions — pick one, read it, propose a run

A reading list for contributors. Pick any paper, read the actual PDF (not a summary), and either
claim the matching task in `papers/` or propose your own 1–2 day experiment from it.
Venue marked so you know what's peer-reviewed vs preprint. Updated 2026-07-08.
